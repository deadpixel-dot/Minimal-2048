# 🎮 Minimal 2048 (Unity Game)

A clean and minimal implementation of the classic 2048 puzzle game built using Unity and C#.

---

## 🚀 Features
- Grid-based tile movement system
- Tile merging logic (2048 mechanics)
- Score tracking system
- Smooth input handling (swipe controls)
- Game state management (win/lose/restart)
- Optimized for Android devices

---

## 🛠️ Tech Stack
- Unity Engine
- C#
- Object-Oriented Programming (OOP)

## 🎯 How to Play
- Swipe in any direction to move tiles
- When two tiles with the same number collide, they merge
- Reach the **2048 tile** to win!
